Movie Studio

Project Status: Completed

Software used: Android Studio

Language used: Kotlin

about this project

Movie Studio is a mobile application developed using Kotlin as part of our "Mobile Game Development" course. The app allows users to search for movies and series, add them to a favorites list, and recommend them to others. Users can also access IMDb ratings and details directly from the app.

The project was built with Firebase for user data management, ensuring each account's information is securely stored and accessible across devices. The app was a collaborative effort, designed to enhance the movie-watching experience with easy search, save, and share features.
