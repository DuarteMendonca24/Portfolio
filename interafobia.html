InTERAFOBIA

PROJECT STATUS: Completed

SOFTWARE USED: Unity

LAnGUAGE USED: C#

ABOUT THIS PROJECT

In Interafobia, you control a character who has a phobia of social interaction. The game features two levels where the objective is to complete a mundane task while avoiding any form of interaction. The game falls under the "Point & Click" genre, with the player using the cursor as the main control, both for movement and item interaction.

This game was developed in 48 hours for the IPCA Game Jam 2023, which had the theme "Interaction." It was created by a team consisting of myself, André Conde, Celestino Machado, and Tiago Fernandes. Interafobia's story is currently available only in Portuguese.
