VENDETTA

Project Status: Completed

Software used: Unity

Language used: C#

About this project

Vendetta is a low-poly FPS game that tells the story of a man seeking revenge for the death of his father. The game has 2 levels and only ends when the vengeance is completed.

This project was the first game I developed using Unity, it was made for the Development of Applied Games course by a group consisting of myself and Francisco Andrade. We worked alongside developing the mechanics for this game, but I was personally responsible for the design of the first level.
